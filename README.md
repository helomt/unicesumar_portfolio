# Portfólio Pessoal — Heloisa Tavares

Landing page de portfólio pessoal desenvolvida como atividade da disciplina de **Programação Frontend**, do curso de **Análise e Desenvolvimento de Sistemas** da **Unicesumar**.

O projeto tem como objetivo aplicar, na prática, conceitos de HTML5, CSS3 e JavaScript puro (sem uso de frameworks), construindo uma página responsiva e interativa para apresentação profissional.

## 🔗 Acesso

Você pode visualizar o projeto abrindo o arquivo `index.html` diretamente no navegador, sem necessidade de servidor ou instalação de dependências.

## 🖥️ Sobre o projeto

A página é dividida nas seguintes seções:

- **Início** — Apresentação com foto de perfil, nome e chamada para ação.
- **Sobre mim** — Resumo da trajetória pessoal e profissional.
- **Experiências** — Histórico de graduação, trabalho e cursos, organizado em abas.
- **Projetos** — Projetos pessoais, colaborações e trabalhos acadêmicos, também organizados em abas.
- **Habilidades** — Grid com as principais tecnologias dominadas.
- **Curiosidades & Links** — Informações complementares e redes sociais.

## ✨ Funcionalidades

- Menu de navegação responsivo, com menu *hamburguer* para telas menores.
- Sistema de abas (*tabs*) para alternar conteúdos nas seções de Experiências e Projetos.
- Botão flutuante de "voltar ao topo".
- Layout responsivo, adaptado para desktop e dispositivos móveis.

## 🛠️ Tecnologias utilizadas

- **HTML5** — Estruturação semântica do conteúdo.
- **CSS3** — Estilização, responsividade (media queries) e efeitos visuais.
- **JavaScript** — Interatividade do menu, sistema de abas e navegação.
- **[Boxicons](https://boxicons.com/)** — Biblioteca de ícones utilizada nas redes sociais.
